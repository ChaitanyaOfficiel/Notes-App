## Run Locally

Clone the project

```bash
  git clone https://github.com/ChaitanyaOfficiel/Notes-App
```

Go to the project directory

```bash
  cd Notes-APP
```


Start the server

```bash
  npm run server
```

run website in local enviroment

```bash
  npm start
```
